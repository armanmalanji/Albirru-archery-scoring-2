# Albirru Archery Scoring

## Instruksi Instalasi PWA  
Untuk menginstal aplikasi ini sebagai Progressive Web App (PWA), ikuti langkah-langkah berikut:  
1. Buka aplikasi di browser Anda.  
2. Klik ikon menu (tiga titik) di sudut kanan atas.  
3. Pilih "Tambahkan ke Layar Utama".  
4. Ikuti instruksi untuk menyelesaikan instalasi.  

## Daftar Fitur  
- Pencatatan skor panahan sederhana  
- Statistik dan analisis skor  
- Antarmuka pengguna yang ramah  
- Mendukung berbagai perangkat  
- Peringatan dan pengingat pengguna  

## Panduan Penggunaan  
1. Masuk ke aplikasi.  
2. Pilih jenis pertandingan panahan.  
3. Masukkan skor setelah setiap ronde.  
4. Tinjau statistik dan analisis setelah pertandingan.

Untuk bantuan lebih lanjut, silakan kunjungi halaman 'Bantuan' di aplikasi.
